1 Foundation

1.1 Utils

1.1.1 translucent

Add alpha channel into rgb color. The origin alpha channel will be override by new value.

translucent($color, $alpha)

  • $color rgba
  • $alpha unit The new value for alpha channel.

1.1.2 be-evne

If $unit is a odd integer than minus one.

be-even($unit)

  • $unit unit

1.1.3 be-odd

If $unit is a even integer than plus one.

be-odd($unit)

  • $unit unit

1.2 Mixins

1.2.1 mix-inner-glow

mix-inner-glow($color, $length)

  • $color rgba
  • $length unit(4) Glow spread length.

1.2.2 mix-outer-glow

mix-outer-glow($color, $length)

  • $color rgba
  • $length unit(4) Glow spread length.

1.2.3 mix-text-highlight

mix-text-highlight($color)

  • $color rgba($c-highlight-light) Highlight color.

1.2.4 mix-text-shadow

Emboss text effect by mix-text-shadow.

mix-text-shadow($color)

  • $color rgba($c-shadow-light) Shadow color.

1.2.5 mix-vinner-shadow

Vertical component inner shadow.

mix-vinner-shadow($deep, $length, $color)

  • $deep unit(2) Shadow deep.
  • $length unit(4) Length of shadow spread.
  • $color rgba($c-shadow-light) Shadow color.

1.2.6 mix-vouter-shadow

Vertical component outer shadow.

mix-vinner-shadow($deep, $length, $color)

  • $deep unit(2) Shadow deep.
  • $length unit(4) Length of shadow spread.
  • $color rgba($c-shadow-light) Shadow color.

1.2.7 mix-hinner-shadow

Horizontal component inner shadow.

mix-vinner-shadow($deep, $length, $color)

  • $deep unit(2) Shadow deep.
  • $length unit(4) Length of shadow spread.
  • $color rgba($c-shadow-light) Shadow color.

1.2.8 mix-houter-shadow

Horizontal component outer shadow.

mix-vinner-shadow($deep, $length, $color)

  • $deep unit(2) Shadow deep.
  • $length unit(4) Length of shadow spread.
  • $color rgba($c-shadow-light) Shadow color.

1.2.9 mix-vgradient

Vertical two color stop gradient. If one of color stop not given gradient will be override into empty.

mix-vgradient($start, $end)

  • $start rgba Start color stop.
  • $end rgba End color stop.

1.2.10 mix-font-face

mix-font-face($name, $path, $weight, $style)

  • $name str Font name
  • $path str Font asset path
  • $weight Font weight
  • $style Font style

1.4.4 clip & ellipsis

1.4.5 bread

1.4.6 Center

Position component according to parent container.

center-*

{* | * ∈ top, right, bottom, left, top-left, top-right, bottom-left, bottom-right}

1.4.7 Amid

Position components amid parent container.

amid-*

{* | * ∈ 45 * n, n ∈ N, n <= 8}

1.4.8 Around

Position components around parent container.

around-*

{* | * ∈ 30 * n, n ∈ N, n <= 12}
